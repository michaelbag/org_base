# Распоряжение РАСП-001

## О проведении обучения по охране труда

В соответствии с требованиями Трудового кодекса Российской Федерации и в целях обеспечения безопасности труда

**РАСПОРЯЖАЮ:**

1. Организовать проведение обучения по охране труда для всех работников организации в срок до 01.03.2024.

2. Назначить ответственным за организацию обучения специалиста по охране труда Петрова Сергея Ивановича.

3. Обучение провести по программе, утвержденной приказом директора.

4. По результатам обучения провести проверку знаний требований охраны труда с оформлением протоколов.

5. Контроль за исполнением настоящего распоряжения возложить на заместителя директора Сидорова А.В.

**Директор**  
Петров Иван Сергеевич
