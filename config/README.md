# Конфигурация

В этой папке хранятся конфигурационные файлы системы.

## config.yaml

Основной файл конфигурации содержит настройки:
- Сервера (хост, порт, режим отладки)
- Документов (директория, поддерживаемые типы)
- PDF генерации (размер страницы, отступы)
- Версионирования (автоматическое отслеживание, интеграция с Git)
- Безопасности (разрешенные расширения, максимальный размер файла)

## Использование

Конфигурация загружается при запуске сервера. Для изменения настроек отредактируйте `config.yaml` и перезапустите сервер.
