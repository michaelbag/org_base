# Тесты

## Запуск тестов

Для запуска всех тестов:

```bash
python -m pytest tests/
```

Или с использованием unittest:

```bash
python -m unittest discover tests
```

## Структура тестов

- `test_document_parser.py` - тесты парсера документов
- `test_version_tracker.py` - тесты системы версионирования

## Добавление новых тестов

При добавлении новых функций создавайте соответствующие тесты в этой директории.
